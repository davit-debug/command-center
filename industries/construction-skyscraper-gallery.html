ხშირად დასმული კითხვები

ცათამბრჯენების ვარიანტებზე — რომელია სწორი არჩევანი, როგორ მუშაობს, რა შეიძლება შევცვალო.

თუ პრემიუმ პრემიუმ დეველოპერი ხართ — Burj Khalifa ან Crowned Spire. თუ მინიმალისტური ხედვა გაქვთ — Glass Curtain ან Modern Tapered. თუ უნიკალური ხართ — Twisted Spiral ან Stacked Boxes. ეჭვი თუ გაქვთ — Stepped Classic ყველაზე უნივერსალურია.
დიახ. ყველა ცათამბრჯენი იყენებს გრადიენტს purple (#A78BFA) → teal (#14B8A6) — brand ფერებს. შეიძლება ფერი შევცვალოთ ნებისმიერ მიმართულებაზე — ოქროსფერი, ცისფერი, ხელით ან თქვენი არჩევანით.
მითხარი რომელი ნომერი მოგწონს (1-10) და ცოცხალ Pyramid გვერდზე ჩავსვამ უკვე არსებული ცათამბრჯენის ნაცვლად. 5 წუთის სამუშაოა.
დიახ — შეიძლება ცათამბრჯენი იყოს სხვადასხვა გვერდზე (Outcomes / Journey / Lifecycle ვარიანტებში). თითოეულ გვერდზე სხვადასხვა ცათამბრჯენი შეიძლება ვცადოთ A/B ტესტისთვის.
SVG (ეს ვარიანტები) უფრო კარგია: (1) მცირე ფაილი (1-2KB ნაცვლად 50-200KB), (2) გადიდებაზე არ ბუნდდება — Retina/4K ეკრანებზე ბრწყინავს, (3) ფერი/ანიმაცია CSS-ით იცვლება, (4) AI-ს კარგად ესმის. Pyramid გვერდი ახლა SVG-ს იყენებს.
Pyramid გვერდის desktop-ზე ცათამბრჯენი გამოიჩინება, მობილურზე — ეკრანის ეკონომიისთვის დამალულია (`hidden lg:flex`). ეს ჩვეულებრივი responsive სტრატეგიაა — hero-ის ვიზუალი desktop-ისთვის, ტექსტი მობილურისთვის.
არა. CSS `drop-shadow` filter უფრო იაფია, ვიდრე SVG-ში `feGaussianBlur` filter. Pyramid გვერდის ცათამბრჯენი იყენებს CSS-ს — Lighthouse score-ში 90+ რჩება. პერფორმანსზე გავლენა — საშუალოდ 1-2ms.
დიახ. SVG-ს `viewBox` ცვლის პროპორციას. ახლა 320×700 (გრძელი). თუ გინდა 320×900 — უფრო გრძელი ცათამბრჯენი. 320×500 — უფრო დაბალი. ან კლასი `max-w-md` → `max-w-lg` ცვლის ეკრანულ ზომას.
დიახ — ცათამბრჯენის წვერზე "იყავი პირველი" ცალკე HTML element-ია (SVG-ის გარეთ). მისი ცვლა მარტივია: ტექსტი, ზომა, ფერი — ნებისმიერი. სხვა ვარიანტი — "გავხდი #1", "ბაზრის ლიდერი" და ა.შ.
დიახ — ფანჯრები ვერტიკალური ხაზებია SVG-ში. შეგვიძლია მეტი დავამატოთ (უფრო რეალისტური) ან წავშალოთ (მინიმალისტური). ახლანდელი 5-7 ხაზი — საშუალო. 12+ — full glass tower. 0 — სრული მინიმალიზმი.
