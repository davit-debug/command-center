როლის მისია

Senior SEO Specialist სრულად ფლობს მასზე მიბმული პროექტების სტრატეგიულ და ოპერაციულ მართვას — Head of SEO-ს მინიმალური ჩარევით. ის პასუხისმგებელია კლიენტის რეალურ შედეგებზე, გუნდის კოორდინაციაზე და პროექტების ხარისხიან, ვადებში ჩაბარებაზე.

SEO სტრატეგია და შესრულება

  • ✓ სრული SEO კვლევა — keyword research, კონკურენტების ანალიზი, ტექნიკური აუდიტი
  • ✓ პროექტის მოქმედების გეგმის შედგენა და განხორციელება
  • ✓ KPI-ების (ტრაფიკი, პოზიციები, ტექნიკური მეტრიკა) ზედამხედველობა

პროექტ მენეჯმენტი

  • ✓ Writers და Junior-ებისთვის მკაფიო, სტრუქტურირებული თასქების გაცემა
  • ✓ ვადების კონტროლი და deliverables-ის ხარისხის გადამოწმება
  • ✓ პროექტის სტატუსის გამჭვირვალე დოკუმენტირება

ტექნიკური გადაწყვეტილებები

  • ✓ Core Web Vitals, crawl budget, log file analysis — დამოუკიდებლად
  • ✓ შესრულებული სამუშაოს ხარისხის ვერიფიკაცია სტანდარტების მიხედვით

კლიენტთან კომუნიკაცია

  • ✓ ყოველდღიური ოპერაციული კომუნიკაცია და სტატუსის გამჭვირვალობა
  • ✓ Reporting შეხვედრებში მონაწილეობა
  • ✓ Upsell შესაძლებლობებისა და churn რისკის დროული იდენტიფიკაცია

მოთხოვნები

  • ✓ 3+ წლის SEO გამოცდილება (საერთაშორისო პროექტები — უპირატესობა)
  • ✓ Ahrefs, GSC, Screaming Frog, Looker Studio — სიღრმისეული ცოდნა
  • ✓ Link building სტრატეგიების დამოუკიდებლად წარმართვა
  • ✓ ქართული და ინგლისური — C1+

პლუსია

  • + E-commerce SEO გამოცდილება
  • + Python / SQL მონაცემთა ანალიზისთვის
  • + გუნდის მართვის ან მენტორინგის გამოცდილება

როლის მისია

Middle SEO Specialist ოპერაციულად ასრულებს SEO პროექტებს სტრატეგიის შესაბამისად და ეტაპობრივად ითვისებს პროექტის სრულ ownership-ს. ის დამოუკიდებლად ახორციელებს on-page ოპტიმიზაციას, mid-level ტექნიკურ SEO-ს და Writers-ის Task Management-ს.

SEO შესრულება

  • ✓ Keyword research, კონკურენტების ანალიზი, on-page ოპტიმიზაცია
  • ✓ Mid-level ტექნიკური SEO დავალებების დამოუკიდებელი შესრულება
  • ✓ კონტენტის ოპტიმიზაცია სტრატეგიის შესაბამისად
  • ✓ Senior/Head-ისგან უკუკავშირის მიღება და გამოყენება ხარისხის გასაუმჯობესებლად

Task Management

  • ✓ Writers-ისთვის მკაფიო, სტრუქტურირებული დავალებების გაცემა და შესრულების კონტროლი
  • ✓ მოქმედების გეგმის ვადებში შესრულების კონტროლი
  • ✓ პროგრესის გამჭვირვალე დოკუმენტირება

კლიენტთან ოპერაციული კომუნიკაცია

  • ✓ რეგულარული აფდეითების მიწოდება შესრულებულ სამუშაოზე
  • ✓ შესრულებული ქმედებებისა და შედეგების დასაბუთება

მოთხოვნები

  • ✓ 1–3 წლის SEO გამოცდილება
  • ✓ Ahrefs, GSC, Screaming Frog-ის ცოდნა
  • ✓ ტექნიკური და on-page SEO-ს მყარი გაგება
  • ✓ ანალიტიკური აზროვნება, შედეგებზე ფოკუსი
  • ✓ ქართული და ინგლისური ენები

პლუსია

  • + Link building გამოცდილება
  • + კონტენტ სტრატეგიის გამოცდილება
  • + საერთაშორისო პროექტებზე მუშაობის გამოცდილება
